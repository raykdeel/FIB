◆ species_distance_update()

void Cjt_Especies::species_distance_update ( bool  indicador,
string  id 
)

Actualiza species_distancies de cada espècie d'en conjunt.

Actualitza table_species depenent de si l'indicador és cert o fals. El primer dels casos indica que s'està afegint una espècie, i per tant, dintre de table_species, les files amb identificador més petit alfabèticament que l'espècie afegida han d'incloure un nou pair<string, double> que emmagatzemi la nova distància. I quan les files de table_species tenen un identificador més gran que el de l'espècie afegida, es crea a la fila del la nova espècie un pair<string, double> amb la nova distància. Si l'identificador és fals, s'elimina de table_species tot parell que inclogui l'identificador de l'espècie eliminada.

Precondició
cert
Postcondició
Actualitza table_species depenent si s'està eliminant un espècie o si s'està afegint.
